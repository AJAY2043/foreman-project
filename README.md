# Where Is My Foreman

A real-time workforce location tracking web application for site managers.

🌐 **Live Demo:** [whereismyforeman.netlify.app](https://whereismyforeman.netlify.app)

---

## About This Project

Built to help construction and field service managers track the real-time 
location of their foremen and field workers. Managers can see where their 
team members are located at any given time via an interactive map interface.

---

## Features

- Real-time location tracking on interactive map
- Firebase real-time database for instant location updates
- Google Maps API integration
- Responsive design for desktop and mobile use
- User authentication

---

## Tech Stack

- **React.js** — frontend framework
- **Firebase** — real-time database and authentication
- **Google Maps API** — interactive map and location display
- **CSS3** — responsive styling
- **Netlify** — deployment
